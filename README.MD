# Internship Management System (IMS)

## Group 6 - BIT Capstone Project

### Project Description
The Internship Management System (IMS) is a web-based application developed to help manage internship groups, student project submissions, and supervisor feedback. The system aims to simplify internship administration and improve communication between students and supervisors.

### Project Objectives
- Provide secure user login.
- Display group information.
- Allow project document submission.
- Support supervisor feedback.
- Track internship project progress.

### Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- XAMPP
